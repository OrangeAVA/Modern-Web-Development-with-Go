# Advanced Concepts of Go



## How to run examples

The examples from this projects can be runned with following command:

```
go run directory/main.go
```

For directory one of the following options can be used:
- channels
- channelsselect
- genericsfunction
- genericstype
- interfaces
- interfacesfunction
- interfacesnil
- interfacesswitch
- methods
- methodsbasictype
- mutex
- panics
- panicstrigger
- waitgroup
