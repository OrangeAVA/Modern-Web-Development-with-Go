# Basic Concepts of Go - Examples



## How to run examples

The examples from this projects can be runned with following command:

```
go run directory/main.go
```

For directory one of the following options can be used:
- deferone
- defertwo
- forloopappend
- forloopcontinuebreak
- forloopomit
- forrangeloop
- functionarg
- functionclosures
- functionone
- functionpointerarg
- functionreturn
- functionslice
- functiontwo
- goplayground
- ifstatement
- maps
- pointers
- switchstatement
- variables
